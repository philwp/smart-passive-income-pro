Smart Passive Income Pro
http://demo.studiopress.com/smart-passive-income/

INSTALL
1. Upload the Smart Passive Income Pro folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Smart Passive Income Pro.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.
